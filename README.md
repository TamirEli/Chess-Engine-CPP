# C++ Chess Engine & Logic Backend

A robust Chess engine implemented in C++ showcasing advanced Object-Oriented Programming (OOP), memory management, and Inter-Process Communication (IPC).

## Background
This project was developed as a final capstone during the **"Magshimim" National Cyber Program**, a prestigious Israeli excellence program for cyber and software engineering.

## Key Technical Features
* **OOP Architecture:** Robust use of inheritance and polymorphism with a base `ChessPiece` class and specialized logic for all chess entities.
* **Complex Game Logic:** Full implementation of chess rules including specialized moves, move validation, and board state management.
* **Inter-Process Communication:** Utilizes Windows Pipes for seamless communication with an external graphical user interface (GUI).
* **Memory Management:** Efficient handling of dynamic memory allocation and pointers.

## Tech Stack
* **Language:** C++
* **Platform:** Windows (utilizing Windows API for Pipes)
* **Status:** Backend Logic & Integration
